# Générateur de carte de visite 

## Intro
Ce programme permet de générer une carte de visite virtuelle chartée selon les différentes entités du groupe **Fides** 
Le principe est simple: saisissez vos informations dans le formulaire, générez votre QR code et scannez-le pour avoir votre carte de visite. 

## Fonctionnalitées
> * Génération d'un QR code 
> * Création d'une page html responsive
> * Changement de couleur en fonction de l'entité
